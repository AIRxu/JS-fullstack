##  端对端  end to end  e2e

##  回归测试
迭代：
v12.1.1 完成开发  完成测试  上线
v12.1.2 完成开发  需要测试v12.1.2和之前的版本
修改之前的代码
同时运行：自己写的测试用例（作用于v12.1.1），自己写测试用例（v12.1.2）
测试用例对应的版本是一劳永逸的，每次回归都可以运行对应版本的测试用例来验证其正确性
